# UstadFinder 🔧

A responsive web app to find skilled workers in Pakistan — electricians, plumbers, AC technicians, painters and carpenters.

Built by **Noor-ul-huda** as Project 1 of the DecodeLabs Full Stack Internship (Batch 2026).

🔗 **Live Site:** [Click here to view](https://your-link-here.netlify.app)

---

## What's in it

- Search workers by name and city
- Filter by service type
- Responsive on mobile and desktop
- Animated hero section
- Contact / job request form

## Built with

- HTML5
- CSS3
- JavaScript (no frameworks)

## How to run

Download the files and open `index.html` in your browser. That's it.
